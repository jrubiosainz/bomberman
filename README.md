# 💣 Bomberman Classic

A classic Bomberman game with cartoon-style graphics, built with **TypeScript**, **HTML5 Canvas**, and **Vite**.

<!-- screenshot -->

## 🎮 Quick Start

```bash
npm install
npm run dev
```

Open http://localhost:5173 in your browser and start playing!

## 🔨 Build & Test

```bash
npm run build   # Production build
npm test        # Run tests
```

## 🎯 Controls

- **Move:** Arrow keys or WASD
- **Place Bomb:** Space

## ✨ Features

- 🎨 Cartoon-style graphics
- 🤖 Enemy AI with smart pathfinding
- 💥 Chain explosions
- 🎁 Powerups: Extra bomb, bigger range, speed boost
- 🏆 Win condition when all enemies defeated

## 🛠️ Tech Stack

- TypeScript
- HTML5 Canvas
- Vite (build tool)
- Vitest (testing)
